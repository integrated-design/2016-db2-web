div タグ, p タグ, h1 タグなどには 横幅や縦幅を指定することが出来ます

div というのは画面上にはなにも現れないけど CSS でレイアウト調整する場合につかえるタグです。
横サイズは width、 縦サイズは height という命令で指定できます。

また margin-left, margin-right を auto と指定することで、左右のマージンを自動調整して、中央寄せにすることが出来ます。
